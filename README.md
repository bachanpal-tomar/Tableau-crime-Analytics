# Crime Analytics Dashboard Project

## Project Overview

This project uses Tableau to analyze crime data and identify crime trends, geographical patterns, and key public safety insights through interactive dashboards.

## Objectives

- Analyze crime patterns and trends
- Identify high-crime locations
- Evaluate crime categories
- Create interactive visual dashboards
- Generate data-driven insights

## Tools Used

- Tableau
- CSV Dataset
- Data Visualization Techniques

## Key Analysis

- Crime Trend Analysis
- Location-Based Crime Analysis
- Crime Category Analysis
- Dashboard Reporting

## Dashboard Features

- Interactive Filters
- Trend Visualizations
- Geographic Analysis
- KPI Monitoring

## Skills Demonstrated

- Tableau

- Dashboard Development
- Exploratory Data Analysis
- Business Intelligence
- Reporting & Insights
